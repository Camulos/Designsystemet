# Slik bruker du produktkortene

Produktkort er ment å brukes som lenker og henvisninger til andre produkter eller livssituasjoner.
Du kan for eksempel henvise til andre stønader som kan være relevant for leseren eller du kan samle alle relevante
produkter innenfor én livssituasjon ved å ta i bruk LargeCard.

## For bruk i Nextjs

Hvis man ønsker å ta i bruk nextjs sin `Link`-komponent kan du gjøre det slik:

```jsx
<Link href="/lenke/til/side" passHref>
  <MicroCard>Er helt eller delvis alene med barn</MicroCard>
</Link>
```

## MicroCard

Dette brukes dersom du allerede har introdusert en stønad eller produkt og ønsker å henvise brukeren til den
konkrete produktsiden (eller livssituasjonssiden).

```
<MicroCard href="/lenke/til/side" type="product">Er helt eller delvis alene med barn</MicroCard>
```
